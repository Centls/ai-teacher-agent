端口配置总结
前端端口：
默认: 3000（Next.js 默认）
设置位置: frontend/package.json - "dev": "next dev --turbopack -p 3000"
启动命令: npm run dev → 运行在 http://localhost:3000
后端端口：
当前: 8004
设置位置: src/server.py - port=8004
前端连接配置: frontend/.env.local - BACKEND_URL=http://127.0.0.1:8004
完整的启动流程：
后端（已启动 ✅）:

.venv/Scripts/python.exe -m src.server
# 运行在 http://0.0.0.0:8004
前端（在 frontend 目录）:

npm run dev
# 运行在 http://localhost:3000
如果你想修改前端端口为其他端口（比如 3001），只需修改 package.json 中的 -p 3000 为 -p 3001 即可。
