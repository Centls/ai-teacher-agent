# Agents Module (智能体模块)

## 简介
本目录包含系统中所有的智能体 (Agents) 实现。每个智能体都是一个独立的 LangGraph 图，具备特定的领域能力。

## 智能体列表

### 1. Marketing Teacher (`marketing/`)
*   **角色**: 营销老师
*   **能力**: 营销策划、文案生成、品牌咨询。
*   **特点**: 具备 CRAG + Self-RAG 能力，支持联网搜索和知识库检索。

### 2. Supervisor (`supervisor/`)
*   **角色**: 总控/路由
*   **能力**: 意图识别、任务分发。
*   **特点**: 作为多智能体系统的入口，负责将用户请求路由给最合适的子智能体。

## 扩展指南
要添加一个新的智能体：
1.  在本目录下创建一个新文件夹 (e.g., `coding/`)。
2.  实现 `graph.py` 定义状态图。
3.  在 `supervisor` 中注册该智能体的能力。
